# Concept → clickable prototype

A repeatable path from idea to something you can put in front of users.

## Steps

1. **Frame** — use the *Problem framing* prompt to sharpen the idea into a job-to-be-done.
2. **Diverge** — generate 5 distinct concepts; pick one with a quick decision matrix.
3. **Structure** — turn the chosen concept into a screen list + low-fi wireframe notes.
4. **Prototype** — generate UI copy and component specs, then build the clickable in Figma.
