# Research synthesizer skill

A packaged Claude skill that takes raw notes and returns affinity-style themes.

## What it does

- Clusters notes into 4–8 themes
- Attaches verbatim quotes as evidence
- Flags contradictions and outliers

## Install

Download [`research-synthesizer.md`](pathname:///skill-files/research-synthesizer.md) and drop it
into your `.claude/skills/` directory, then reload Claude Code.
