# Jira Worklog Skill

A Claude skill that lets you manage Jira worklogs using natural-language prompts, so logging time stops being a chore you forget.

## Download

[⬇ Download the skill file (.skill)](pathname:///resource-files/jira-worklog-skill/jira-update_skill_file.skill)

> A `.skill` file is a zip bundle. Install it into your Claude skills directory, then reload Claude.

## The problem it solves

- Worklogs are often updated late or missed entirely.
- Managers end up spending time following up with team members.
- Reporting and status visibility become less reliable.

## What it does

A Jira Worklog Skill that lets you manage worklogs through simple prompts instead of clicking through Jira.

## Current capabilities

- Add Jira worklogs (day / week / month) with a simple prompt
- Modify existing worklogs
- Generate worklog summaries for any date range
